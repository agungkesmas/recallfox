📋 Cara dapat API Key (klik untuk lihat)

Groq (recommended, free, fast) — sudah pre-configured

API key Groq sudah diisi sebagai default (lihat field API Key). Groq hosting model Llama 3.3 70B gratis dengan rate limit generous.

  1. Buka https://console.groq.com/keys & login (gratis).
  2. Klik Create API Key, salin key (format gsk_...).
  3. (Opsional) Ganti key default di atas dengan key Anda sendiri untuk privasi.
  4. Model gratis populer: llama-3.3-70b-versatile, llama-3.1-8b-instant, mixtral-8x7b-32768.

Google Gemini (free tier) — untuk fallback

  1. Buka https://aistudio.google.com/app/apikey & login dengan Google.
  2. Klik Create API Key, pilih project (atau buat baru).
  3. Salin key (format AIza...).
  4. Paste ke field Fallback API Key di atas.
  5. Model gratis: gemini-2.0-flash (default), gemini-1.5-flash, gemini-1.5-flash-8b.
  6. Rate limit free tier: 15 RPM, 1500 RPD — cukup untuk asisten pribadi.

xAI Grok

  1. Buka https://console.x.ai & login.
  2. Buat API key (format xai-...).
  3. Paste ke field API Key. Model: grok-3, grok-3-mini, grok-2.

z.ai (GLM-4.6)

  1. Buka https://z.ai/manage/apikey & login.
  2. Create API Key. Model: glm-4.6.

OpenAI

  1. Buka https://platform.openai.com/api-keys.
  2. Create new secret key (format sk-...). Model: gpt-4o-mini.

Custom (OpenAI-compatible)

Untuk endpoint self-hosted atau provider lain (Ollama, LM Studio, Together, Anyscale, dll). Isi Base URL + API key + model name.

⚠️ Keamanan API key: API key disimpan lokal di browser.storage.local. Key Groq default di kode ini adalah key pribadi developer — segera ganti dengan key Anda sendiri di Settings untuk keamanan. Jangan share file zip addon ini karena berisi key Groq.

⚙️ Toppings (Orchestrator)

Toppings adalah behavior modifier yang nempel di prompt saat diinject ke AI — seperti "topping" pada makanan. Pilih toppings di editor prompt, dan saat Anda klik "Sisipkan", toppings otomatis di-append ke body prompt. Bisa multiple, stacking. Auto-aktif saat inject (bisa di-toggle per item).

💡 12 built-in toppings tersedia. Custom toppings (bikin sendiri) ada di roadmap. Toppings disimpan per-item — edit item mana saja untuk menambah/menghapus toppings.

📚 Bantuan & Dokumentasi

Penjelasan singkat setiap fitur RecallFox. Klik tab di bawah untuk lihat detail.

🦊 Apa itu RecallFox?

RecallFox adalah addon Firefox untuk menyimpan prompt & konteks AI dengan satu klik. Local-first (data di device Anda), sync opsional via Firefox Sync, backup terenkripsi. Mendukung 21+ AI tool: z.ai, ChatGPT, Claude, Gemini, DeepSeek, Qwen, Kimi, dan lainnya.

💾 Vault (Penyimpanan)
  1. Buka vault: klik ikon RecallFox di toolbar (popup)
  2. 5 tab: Semua / Prompt / Konteks / Snapshot / Link / Bundle + tombol Asisten di header.
  3. Search: ketik di search bar, atau tekan / untuk fokus cepat.
  4. Klik item = inject ke AI textarea. Klik ganda = edit.
  5. Counter di tiap tab menunjukkan jumlah item.
✍️ Simpan Prompt / Konteks
  1. Blok teks di halaman web mana saja.
  2. Klik kanan → "Simpan sebagai Prompt" atau "Simpan sebagai Konteks".
  3. Toast konfirmasi muncul di pojok kanan bawah.
  4. Item otomatis tersimpan ke vault dengan URL sumber & timestamp.
🔗 Simpan Halaman / Link
  1. Halaman yang sedang dibuka: klik kanan di mana saja → "Simpan Halaman Ini".
  2. Link di halaman: klik kanan pada link → "Simpan Link Ini".
  3. Item tersimpan sebagai tipe Link dengan URL + judul otomatis.
  4. Klik item Link di vault = buka di tab baru.
📸 Snapshot Percakapan AI
  1. Saat di halaman AI (z.ai, ChatGPT, dll), klik kanan → "Ambil Snapshot Percakapan".
  2. Atau tekan Ctrl+Shift+3.
  3. Atau klik tombol mengambang 📸 di pojok kanan bawah halaman AI.
  4. Modal muncul dengan ringkasan otomatis. Edit judul + tag, lalu simpan.
  5. Snapshot menyimpan 50 pesan terakhir (teks saja, maks 10KB).
📦 Bundle (Project Session)
  1. Bundle = gabungan beberapa item (prompt + link + konteks) jadi satu paket.
  2. Klik bundle = buka semua link di tab baru + inject prompt ke AI textarea.
  3. Cocok untuk project session: GitHub repo + Supabase + Vercel + prompt checklist.
  4. Untuk saat ini, bundle dibuat via edit manual item atau import dari backup. UI builder visual ada di roadmap.
⚙️ Variabel di Prompt
  1. Tulis {{nama_variabel}} di body prompt.
  2. Saat diinject, modal muncul untuk isi nilai variabel.
  3. Contoh: Review kode {{bahasa}} untuk project {{project}}
  4. Editor vault otomatis detect variabel & tampilkan sebagai chips.
🍕 Toppings (Orchestrator)
  1. Buka editor item (klik ganda item atau tombol Edit).
  2. Di field Toppings, klik "+ Pilih topping".
  3. Pilih satu atau beberapa topping dari popover (12 built-in tersedia: Research Mode, Step-by-step, Deep Think, Brainstorm, Critic Mode, Simplifier, Translator, Code Reviewer, Checklist Maker, Iterative Improver, Expert Persona, Output Format).
  4. Toppings yang dipilih muncul sebagai chips — klik × untuk hapus.
  5. Save item. Saat Anda klik "Sisipkan", toppings otomatis di-append ke body prompt.
  6. Item dengan toppings aktif menampilkan badge ⚙ topping-id di kartu item.
📎 Inject dengan Lampiran Link
  1. Klik tombol 📎 di samping tombol "Sisipkan" pada item prompt/konteks/snapshot.
  2. Modal muncul: pilih link mana yang mau dilampirkan (bisa multiple).
  3. Pilih posisi (atas/bawah prompt) dan edit teks pengantar.
  4. Preview live teks final sebelum diinject.
  5. Format output: - [Title](URL) (markdown link, otomatis di-render AI).
⚡ Quick Switch AI
  1. Klik tombol ⚡ AI di header vault.
  2. Dropdown muncul dengan 21 AI tool dalam 3 grup: Lokal / Barat / China.
  3. Klik Open untuk buka AI tool di tab baru.
  4. AI tool yang sedang aktif ditandai badge "⚡ aktif".
🌙 Dark / Light Mode
  1. Klik tombol ☀️/🌙 di header vault untuk toggle cepat.
  2. Atau atur permanen di Settings → "Tema tampilan" (Auto / Terang / Gelap).
  3. Default: Auto — ikuti system preference OS Anda.
🔐 Backup & Sync
  1. Backup lokal: Settings → "Backup Lokal" → Export (plain JSON atau terenkripsi .rfvault).
  2. Enkripsi pakai AES-GCM 256 + PBKDF2 (100k iter) via Web Crypto API native.
  3. Firefox Sync: Settings → "Sinkronisasi" → aktifkan untuk sync antar device Firefox account.
  4. Sync di-chunk 90KB per key (limit Firefox Sync 100KB/key).
  5. Konflik resolve: item-level last-write-wins by updatedAt.
🤖 AI Assistant
  1. Setup API key di section AI Assistant di atas.
  2. Buka sidebar → tab Asisten.
  3. Tanya apa saja tentang cara pakai RecallFox — asisten paham semua fitur.
  4. Contoh: "gimana cara simpan link?", "apa itu bundle?", "shortcut untuk snapshot?"
  5. Riwayat chat disimpan lokal, bisa di-clear kapan saja.
